Work Page — 3 Ansätze

Alle nutzen das Design-Vokabular der Projektseiten (section-labels, badges, gleiche Typografie). Klick auf deinen Favoriten.

All Colorist DP

A — Badge-Filter links

Section-Label "Work" oben links, darunter die Kategorien als badge / badge--outline — exakt wie auf den Projektseiten. Linksbündig, kein zentriertes UI-Element. Filter fühlt sich an wie ein natürlicher Teil der Seite.

Work
All Colorist DP

B — Inline-Nav mit Label

Section-Label "Work" und Kategorien in einer Zeile, getrennt durch Abstand. Aktive Kategorie in Weiß, Rest in Grau. Wie eine Sub-Navigation, die durch die border-bottom-Linie an die section-labels anknüpft. Könnte sticky werden beim Scrollen.

Selected Projects
All Colorist DP

C — Header + Badge-Filter rechts

Section-Label oben, darunter eine Zeile mit Heading links ("Selected Projects") und Badge-Filter rechts. Gibt der Seite mehr Gewicht und Kontext. Badges in badge--muted statt outline — dezenter, aktiver Filter in Accent.

Gemeinsam bei allen Ansätzen

  • Section-Label "Work" wie auf Projektseiten
  • Filter-Elemente nutzen Badge-Styles statt generische Buttons
  • Linksbündig oder strukturiert — nicht mehr zentriert/freistehend
  • max-width: 900px wie Projektseiten-Content

Unterschiede

  • A — Minimalistisch, nah am Projektseitenflow
  • B — Nav-artig, könnte sticky werden, sehr clean
  • C — Mehr visuelles Gewicht, "rahmt" die Seite ein
