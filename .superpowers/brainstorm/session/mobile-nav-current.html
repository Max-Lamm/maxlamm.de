Issues with Current Approach

Hide on scroll down — The nav disappears when scrolling down. User has to scroll up to get it back. Easy to lose, disorienting.
No context — Just ‹ and › arrows. No project name, no indication of what's next or where you are in the portfolio.
Bottom bar competes with browser UI — On iOS Safari / Chrome, the bottom bar sits right where the browser's own controls are. Accidental taps, visual clutter.
No "back to grid" shortcut — To return to /work/, user must scroll all the way up to the nav bar.
