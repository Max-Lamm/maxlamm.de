Source Media Workflow

Flat Quicktime Workflow
{{ if eq .Lang "en" }}A flattened Quicktime serves as the basis – simpler, but with limitations.{{ else }}Ein geflattetes Quicktime dient als Basis – einfacher, aber mit Einschränkungen.{{ end }}
{{ if eq .Lang "en" }}Prepare sequence{{ else }}Sequenz vorbereiten{{ end }}
01
{{ if eq .Lang "en" }}Create a copy of your sequence and add a descriptive suffix, e.g. forColor.{{ else }}Erstelle eine Kopie deiner Sequenz und füge einen beschreibenden Suffix hinzu, z.B. forColor.{{ end }}
02
{{ if eq .Lang "en" }}Remove graphics, text overlays and other elements not relevant for grading.{{ else }}Grafiken, Texteinblendungen, Overlays und andere nicht für's Grading relevante Elemente entfernen.{{ end }}
{{ if eq .Lang "en" }}Exception:{{ else }}Ausnahme:{{ end }} {{ if eq .Lang "en" }}If I am to create the final exports in Resolve, leave all elements unchanged.{{ else }}Falls ich die finalen Exporte in Resolve erstellen soll, lass alle Elemente unverändert.{{ end }}
03
{{ if eq .Lang "en" }}Reduce the clips and edits to as few video tracks as possible – ideally just 1–2. When finished, there should be no clips or clip sections in the sequence that are not visible in the final cut.{{ else }}Reduziere die Clips und Bearbeitungen auf so wenige Videospuren wie möglich – idealerweise nur 1–2. Nach Abschluss sollten keine Clips oder Clip-Bereiche in der Sequenz vorhanden sein, die im finalen Schnitt nicht sichtbar sind.{{ end }}
04
{{ if eq .Lang "en" }}Join all through edits, unless they serve to subdivide speed changes or reframes.{{ else }}Verbinde alle durchgehenden Edits, es sei denn, sie dienen der Unterteilung von Geschwindigkeitsänderungen oder Reframes.{{ end }}
05
{{ if eq .Lang "en" }}Replace all multi-cam clips, nested sequences or compound clips with their source clips.{{ else }}Ersetze alle Multi-Cam-Clips, verschachtelte Sequenzen oder Compound Clips durch ihre Quellclips.{{ end }}
06
{{ if eq .Lang "en" }}Swap all proxies or transcoded clips for the original source clips.{{ else }}Tausche alle Proxies oder transkodierten Clips gegen die Original-Quellclips aus.{{ end }}
{{ if eq .Lang "en" }}Clean up{{ else }}Aufräumen{{ end }}
07
{{ if eq .Lang "en" }}Delete empty video tracks.{{ else }}Leere Videospuren löschen.{{ end }}
08
{{ if eq .Lang "en" }}Delete all audio tracks and clips. Replace with an audio mixdown.{{ else }}Alle Audiospuren und Clips löschen. Und durch einen Audio-Mixdown ersetzten.{{ end }}
{{ if eq .Lang "en" }}Export & Handoff{{ else }}Export & Übergabe{{ end }}
09
{{ if eq .Lang "en" }}Copy all original media to a working drive.{{ else }}Kopiere alle Originalmedien auf ein Arbeitslaufwerk.{{ end }}
10
{{ if eq .Lang "en" }}Export an XML or AAF of the sequence.{{ else }}Exportiere eine XML oder AAF der Sequenz.{{ end }}
11
{{ if eq .Lang "en" }}Export a Quicktime of the sequence in native resolution as ProRes LT or H.264.{{ else }}Exportiere ein Quicktime Video der Sequenz in nativer Auflösung als ProRes LT oder H.264.{{ end }}
{{ if eq .Lang "en" }}Prepare sequence{{ else }}Sequenz vorbereiten{{ end }}
01
{{ if eq .Lang "en" }}Create a copy of your sequence and add a descriptive suffix, e.g. forColor.{{ else }}Erstelle eine Kopie deiner Sequenz und füge einen beschreibenden Suffix hinzu, z.B. forColor.{{ end }}
02
{{ if eq .Lang "en" }}Make sure the sequence resolution matches the desired final delivery resolution.{{ else }}Stelle sicher, dass die Sequenzauflösung mit der gewünschten finalen Auslieferungsauflösung übereinstimmt.{{ end }}
03
{{ if eq .Lang "en" }}Verify that all clips fill the frame completely without black bars (pillarboxing or letterboxing).{{ else }}Überprüfe, dass alle Clips das Bild vollständig ausfüllen, ohne schwarze Balken (Pillarboxing oder Letterboxing).{{ end }}
04
{{ if eq .Lang "en" }}Remove all dissolves or other transitions.{{ else }}Entferne alle Überblendungen oder andere Übergänge.{{ end }}
05
{{ if eq .Lang "en" }}Finalise any position or zoom adjustments, as these will be baked into the colour grade.{{ else }}Schließe Anpassungen an Positionen oder Zooms ab, da diese in die Farbkorrektur gebacken werden.{{ end }}
06
{{ if eq .Lang "en" }}Remove graphics, text overlays and other elements not relevant for grading.{{ else }}Grafiken, Texteinblendungen, Overlays und andere nicht für's Grading relevante Elemente entfernen.{{ end }}
{{ if eq .Lang "en" }}Exception:{{ else }}Ausnahme:{{ end }} {{ if eq .Lang "en" }}If you want to create the final exports in Resolve, leave all elements unchanged and export them as a separate Quicktime with alpha (without footage!).{{ else }}Wenn du die finalen Exporte in Resolve erstellen möchtest, lass alle Elemente unverändert und exportiere sie als separates Quicktime mit Alpha (ohne Footage!).{{ end }}
07
{{ if eq .Lang "en" }}Remove filters and LUTs.{{ else }}Filter und LUTs entfernen.{{ end }}
08
{{ if eq .Lang "en" }}Swap all proxies or transcoded clips for the original source clips.{{ else }}Tausche alle Proxies oder transkodierten Clips gegen die Original-Quellclips aus.{{ end }}
{{ if eq .Lang "en" }}Export & Handoff{{ else }}Export & Übergabe{{ end }}
09
{{ if eq .Lang "en" }}Export an EDL, XML or AAF of the sequence.{{ else }}Exportiere eine EDL, XML oder AAF der Sequenz.{{ end }}
10
{{ if eq .Lang "en" }}Export a Quicktime of the sequence in native resolution as ProRes 4444.{{ else }}Exportiere ein Quicktime der Sequenz in nativer Auflösung als ProRes 4444.{{ end }}

{{ if eq .Lang "en" }}Limitations of this workflow{{ else }}Einschränkungen dieses Workflows{{ end }}

  • {{ if eq .Lang "en" }}Dissolves or other transitions in the sequence cannot be carried over. These must be manually re-added afterwards.{{ else }}Überblendungen oder andere Übergänge in der Sequenz können nicht übernommen werden. Diese müssen im Nachhinein manuell wieder hinzugefügt werden.{{ end }}
  • {{ if eq .Lang "en" }}The final render resolution is limited to the resolution of the delivered flat Quicktime, regardless of the original footage resolution.{{ else }}Die finale Renderauflösung ist auf die Auflösung des gelieferten Flat-Quicktime beschränkt, unabhängig von der ursprünglichen Footage-Auflösung.{{ end }}
  • {{ if eq .Lang "en" }}All composited elements will be treated as a single flat element.{{ else }}Alle zusammengesetzten Elemente werden als ein einzelnes flaches Element bearbeitet.{{ end }}
